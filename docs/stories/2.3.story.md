# Story 2.3 — Sincronização de Dados

**Epic:** 2 — Meta Ads Integration
**Status:** Aguardando 2.1 e 2.2
**Estimativa:** 3h

## Descrição

Implementar o engine de sincronização que busca campanhas, conjuntos e anúncios da Meta API e faz upsert no Supabase. Endpoint `/api/sync` é o coração do sistema.

- [ ] Implementar `POST /api/sync/route.ts`:

  - [ ] Buscar métricas dos últimos 30 dias por entidade
